Plugins

Two types: pattern and search.

Pattern Plugin

Filters results by URL glob pattern.

{
  "name": "Wiki",
  "prefix": "w",
  "pluginType": "pattern",
  "patterns": ["docs.example.com"],
  "color": "#f38ba8"
}

Search Plugin

Redirects to a URL with %s replaced by query.

{
  "name": "CodeSearch",
  "prefix": "cs",
  "pluginType": "search",
  "url": "https://grep.app/search?q=%s",
  "color": "#fab387"
}

Prefixes

First word followed by a space activates a prefix. A label appears in the search bar when active.

"prefixes": {
  "history": "h",
  "tabs": "t",
  "bookmarks": "b"
}

Glob Syntax

Matched against hostname + path (no protocol).

SymbolMatches
*Any chars except . and /
**Any chars including . and /

Domain-only patterns (no /) auto-match all paths.

PatternMatches
github.comgithub.com/user/repo
*.github.comgist.github.com/**
docs.example.com/wiki/view/*One level under view/
docs.example.com/wiki/view/**Any depth under view/

Source Colors

Customize badge colors for built-in types.

"sourceColors": {
  "tabs": "#89b4fa",
  "bookmarks": "#f9e2af",
  "history": "#a6e3a1"
}

Shortcut

Modifier keys + key name.

"shortcut": {
  "ctrlKey": false,
  "metaKey": true,
  "key": "k"
}
